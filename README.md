# The Next Web Clone

    This project consists of replicating the The Next Web website putting emphasis on how it behaves
    differently depending on the size of the screen where the website is rendered.

![screenshot](./screenshot.png)

## :package: Built With

    - Plain HTML5
    - Pure CSS3
    - Font Awesome
